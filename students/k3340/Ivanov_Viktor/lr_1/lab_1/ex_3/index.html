Сервер успешно отдает HTML-страницу через сокеты

Что реализовано:

  • HTTP-сервер на чистом Python с использованием сокетов
  • Загрузка HTML-страницы из файла index.html
  • Корректные HTTP-заголовки и статус-коды
  • Обработка HTTP-запросов от браузера
  • Красивый и современный дизайн страницы

Технические детали:

Протокол: HTTP/1.1

Транспорт: TCP сокеты

Порт: 8082

Библиотека: socket

# Пример HTTP-ответа сервера:
HTTP/1.1 200 OK
Content-Type: text/html; charset=UTF-8
Content-Length: [размер файла]
Connection: close

[HTML содержимое]

Создано с использованием Python и библиотеки socket
